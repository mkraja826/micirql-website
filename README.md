# Micirql Website

Micirql is a software company founded by Karthik Raja that develops Android apps, landing pages, dashboards, and business software.

## Pages / Sections

- Home
- About Us
- Founder Page
- Micirql App Store
- Services
- Contact

## Tech Stack

- React
- Vite
- Custom CSS
- lucide-react icons

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
